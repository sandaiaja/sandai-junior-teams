## 更新方法

1. ファイルを編集
2. 変更をコミット
   ```bash
   git add .
   git commit -m "更新内容"
   ```
3. GitHubにプッシュ
   ```bash
   git push origin master
   ```

プッシュ後、数分でGitHub Pagesに反映されます。
